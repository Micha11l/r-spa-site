# Documentation

This folder contains project documentation, deployment guides, and development notes.

## Files

- **DEPLOYMENT_CHECKLIST.md** - Complete deployment checklist for production
- **ENV_COMPARISON.md** - Environment variables comparison (local vs production)
- **VERCEL_SETUP_GUIDE.md** - Step-by-step Vercel deployment guide
- **QUICK_START.md** - Quick start guide for developers
- **IMPROVEMENTS_SUMMARY.md** - Summary of improvements and changes
- **SECURITY_FIXES.md** - Security fixes and best practices

## Note

Sensitive files (API keys, passwords, production secrets) are NOT stored here.
They are excluded via `.gitignore` in the project root.
