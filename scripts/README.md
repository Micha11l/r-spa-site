# Scripts

This folder contains development and testing scripts.

## Files

- **browser_diagnose.js** - Browser-based diagnostic script
- **check-components.sh** - Component validation script
- **diagnose_amount_issue.sh** - Amount handling diagnostic script
- **fix-client-manual.sh** - Manual client fix script
- **fix-client-usage.sh** - Client usage fix script
- **verify-updates.sh** - Update verification script

## Usage

These scripts are for development and testing purposes only.
They should NOT be run in production environments.

Make scripts executable before running:
```bash
chmod +x scripts/*.sh
```
